# Artifact Templates

This directory contains canonical JSON templates for round artifacts.

Templates currently provided:

- `metrics.template.json`
- `decision.template.json`
- `claude_report.template.json`
- `verifier_report.template.json`
- `draft_data_request.template.json`

Use these templates as starting points for real round outputs. Replace example values, but keep the keys stable unless the harness docs are explicitly updated.
