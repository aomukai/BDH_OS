# Teacher Skills

Each markdown file in this directory defines one intervention the teacher may use.
These files are intended to stay:

- legible
- patchable
- versioned
- auditable

## Required Skill Shape

Every skill should define:

- purpose
- when to use
- inputs
- execution steps
- success metrics
- stop conditions
- what to log
- likely next intervention if it fails
